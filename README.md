# todo-list-web
AoCubo Challenge

## Localhost
Para rodar essa aplicação no seu localhost você precisa ter instalado Node.JS.

Execute:
```
1- Git clone repository
2- yarn install
3- yarn start
4- Check your localhost:3000
```
